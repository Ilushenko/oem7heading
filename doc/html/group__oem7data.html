Detailed Description

Provide OEM7 Protocol structures and data.

Classes, structures and enumerators that ensure the operation of NovAtel OEM7 dual-antenna receiver

Note
All structures must defined with 1-byte alignment
  • Use __attribute__((__packed__)) by GCC
  • Use #pragma pack(push,1) and #pragma pack(pop) directives by Win32
